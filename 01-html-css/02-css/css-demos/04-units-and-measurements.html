CSS Units and Measurements Practice

What are CSS Units?

CSS units define how lengths, sizes, and spaces are measured in web design. They can be absolute or relative.

Common CSS Units

  • px (pixels): Absolute unit, fixed size regardless of other settings.
  • % (percent): Relative to parent element's size.
  • rem (root em): Relative to the root element's font size (usually <html>).
  • em: Relative to the parent element's font size.
  • vw / vh: Relative to viewport width / height.

Difference between rem and em

rem depends on the root element's font size. For example, if the root font size is 16px, then 1rem = 16px. This means rem units are consistent throughout the page.

em depends on the font size of the parent element. It can compound if nested elements use em for size, causing sizes to multiply. This makes em more flexible but sometimes tricky.

Example: If a parent has font-size: 20px;, then 1em = 20px for its children. For the children’s children, if they use 1em, it is relative to their direct parent's font size.
