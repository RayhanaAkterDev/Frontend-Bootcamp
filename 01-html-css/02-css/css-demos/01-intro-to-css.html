Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style HTML elements visually.

There are 3 types of CSS:

  • Inline – using style attribute directly in tags
  • Internal – using <style> tag inside <head>
  • External – linking a separate .css file
